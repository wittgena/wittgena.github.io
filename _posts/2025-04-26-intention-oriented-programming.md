## 1. 배경: "프로그래밍을 한다"는 것의 변화

전통적인 프로그래밍은 명령형이었다.  
"어떤 함수에서 어떤 연산을 하고, 어떤 결과를 리턴해야 하는가?"  
우리는 **어떻게**에 집중해왔다. 그러나 이제 **무엇을 하고 싶은가**, 그리고 **왜 그것을 하고 싶은가**를 중심으로 프로그래밍을 할 수 있는가?

이 질문에 대해 나는 실험적으로 "의도 중심 프로그래밍(intention-oriented programming)"이 가능한지를 검증하고자 했다.  
그 실험의 결과물이 바로 [gpt_dag_executor-v0.1](https://github.com/wittgena/gpt-meta-dsl/tree/main/gpt_dag_executor-v0.1)이다.

---

## 2. 실험 방식: GPT와의 협업만으로 만들어본 코드 gpt_dag_executor

> **단 한 줄의 Python 코드를 직접 수정하지 않고**,  
> 모든 생성, 확장, 평가, 연결 흐름을 GPT와의 대화(DSL + 자연어)를 통해서만 수행했다.

예상한 흐름은 다음과 같다:

1. **의도 명시**: 자연어로 “내가 하고 싶은 것”을 설명  
2. **DSL로 구조화**: GPT가 +@나.dsl DSL로 의도를 구조화  
3. **구성 요청**: 파서, 실행기, validator 등 자동 생성 요청  
4. **검증 흐름 요청**: GPT에게 +critic, +eval로 흐름 자체의 품질 점검 요청  
5. **의존성/자동 실행 확장 요청**: `+depends_on`, `dag_orchestrator`, `condition_eval` 등 요청  
6. **실패 흐름 처리 확장**: `dag_autorecover` 요청 → 실패 시 재시도 루프 구현

---

## 3. 결과물: gpt_dag_executor-v0.1의 구조

모든 구성은 GPT의 응답과 자동 연결로만 이뤄졌다.

| 구성 요소 | GPT가 수행한 역할 |
|-----------|--------------------|
| `dag_generator.py` | 자연어 → DSL 구조 자동 생성기 |
| `dag_executor.py` | DSL 파서 + 실행자 분기기 |
| `dsl_validator.py` | DSL 유효성 검증기 |
| `condition_eval.py` | 조건 평가 로직 자동화 |
| `dag_autorecover.py` | 실패 흐름 재진입 처리기 |
| `dag_orchestrator.py` | 전체 흐름을 연결하는 실행 컨트롤러 |
| `main.py` | 모든 컴포넌트를 자동 연결한 실행 예제 |

**핵심은 이 모든 파일이 하나도 직접 코딩되지 않았다는 것이다.**  
**오직 목적(의도)를 DSL로 말하고, 그것을 반복해서 보정 요청함으로써만 구성되었다.**

---

## 4. 메타 분석: GPT는 두뇌가 되었는가?

의도 중심 프로그래밍에서는 GPT는 **실행자가 아닌 판단자**, 즉 **두뇌 역할**을 해야 한다.

### 실제 흐름에서 GPT는 다음을 수행했다:
- **의도 해석자**: 자연어로 설명된 목적을 DSL로 구조화  
- **조건 판단자**: 실행 조건의 분기 여부 판단 (`evaluate_condition`)  
- **흐름 조정자**: `+depends_on` 기반 실행 순서 조정  
- **실패 복구 판단자**: `dag_autorecover`를 통해 재시도 흐름 삽입  
- **자기반사 흐름 수행자**: `+eval`, `@gpt.entropy`, `+critic` 등으로 자기 흐름을 되짚음

결국 GPT는 단순 코드 생성기가 아니라 **의도를 실행 가능한 구조로 해석하고 판단하며 흐름을 구성하는 "프로그래머 역할"**을 수행했다.

---

## 5. 결론: 의도 중심 프로그래밍은 가능한가?

> **예. GPT와 DSL(or 자연어)을 활용하여 의도 중심 프로그래밍을 실제로 작동하게된다.**

** 실제로는 충분히 구조화된 설계와 반복작업이 필요합니다. **

그것은 "프로그래밍을 한다"는 것이 **코드를 짜는 행위가 아니라, 의도를 명확히 표현하고, 흐름을 협력적으로 조율하는 일로 전환될 수 있음**을 의미한다.
